## Learning Objectives
After this module, you will be able to:
- Execute prompt injection attacks (direct and indirect)
- Identify jailbreaking techniques and their variants
- Extract training data and system prompts from LLMs
- Understand model stealing through query APIs
- Identify supply chain vulnerabilities in AI pipelines
- Poison RAG knowledge bases with malicious content
- Exploit agentic systems through tool manipulation
- Craft adversarial examples against ML models

## ⚠️ Ethical Use Notice
These techniques are taught for defensive purposes. Always obtain proper authorization before testing AI systems you don't own.
