## Learning Objectives
After this module, you will be able to:
- Apply security-by-design principles to AI projects
- Conduct threat modeling sessions for AI applications
- Implement secure development practices for LLM apps
- Design security test suites for AI systems
- Audit and secure AI supply chains (models, data, packages)
- Conduct security reviews and assessments of AI deployments
