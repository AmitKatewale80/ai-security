# AI Security Tools

Open source and commercial tools for testing, defending, and monitoring AI systems.

## Overview

This reference catalogs key AI security tools organized by function. Most are open source and actively maintained.

## Offensive / Red Teaming Tools

| Tool | Purpose | Link |
|------|---------|------|
| Garak | LLM vulnerability scanner (probes, detectors) | https://github.com/leondz/garak |
| PyRIT | Microsoft's AI red team automation framework | https://github.com/Azure/PyRIT |
| Counterfit | Adversarial ML attack simulation | https://github.com/Azure/counterfit |
| ART | IBM Adversarial Robustness Toolbox | https://github.com/Trusted-AI/adversarial-robustness-toolbox |
| TextAttack | NLP adversarial attack framework | https://github.com/QData/TextAttack |
| PromptMap | Prompt injection testing | https://github.com/utkusen/promptmap |

## Defensive Tools

| Tool | Purpose | Link |
|------|---------|------|
| NeMo Guardrails | Programmable LLM safety rails | https://github.com/NVIDIA/NeMo-Guardrails |
| Guardrails AI | Output validation framework | https://github.com/guardrails-ai/guardrails |
| Rebuff | Prompt injection detection | https://github.com/protectai/rebuff |
