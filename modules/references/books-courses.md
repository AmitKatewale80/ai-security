# Books & Courses on AI Security

Recommended books and online courses for deepening AI security knowledge.

## Overview

These resources provide structured learning paths from introductory to advanced levels. Courses are selected for practical applicability and instructor credibility.

## Books

| Title | Author(s) | Year | Focus |
|-------|-----------|------|-------|
| Adversarial Machine Learning | Kurakin, Goodfellow, Bengio | 2023 | Comprehensive adversarial ML textbook |
| AI and Machine Learning for Coders | Laurence Moroney | 2020 | ML fundamentals (security context) |
| The Alignment Problem | Brian Christian | 2020 | AI safety and alignment |
| Not with a Bug, But with a Sticker | Ram Shankar Siva Kumar, Hyrum Anderson | 2024 | Practical adversarial ML |
| Security Engineering (4th ed.) | Ross Anderson | 2020 | Security fundamentals (AI chapters) |
| Practical Deep Learning for Cloud, Mobile & Edge | Koul, Ganju, Kasam | 2019 | Deployment security |

## Free Resources

- **OWASP AI Exchange** — Community wiki on AI security

- **AI Village Talks** — DEF CON AI security presentations
- **Hugging Face Security Blog** — ML platform security

## Learning Path Recommendation

1. Start with ML fundamentals (understand what you're securing)
2. Study OWASP Top 10 for LLM Applications
3. Complete a hands-on course on adversarial ML
4. Practice with Garak and PyRIT tools
5. Read key papers (see [papers.md](papers.md))
6. Engage with communities (see [communities.md](communities.md))
