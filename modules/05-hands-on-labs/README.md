# Module 5: Hands-On Labs

Practice everything you've learned with realistic, airline-themed security labs.

## Labs

All hands-on labs are located in the **airline-labs** directory:

👉 **[Go to Labs →](../../airline-labs/)**

## Lab Overview

The airline-labs folder contains 20+ hands-on labs covering:
- Supply chain attacks on AI models
- Model stealing and cloning
- Chatbot hijacking via prompt injection
- RAG data extraction
- Malicious code injection in models
- Model signing and verification
- AI-powered phishing
- And many more...

## How to Use

1. Navigate to the [airline-labs](../../airline-labs/) directory
2. Each lab has its own folder with a README, requirements, and scripts
3. Labs are designed to be run locally with Python
4. Each lab demonstrates both the attack AND the defense

## Prerequisites
- Modules 1-4 (recommended but not required)
- Python 3.9+
- See individual lab requirements.txt files for dependencies
