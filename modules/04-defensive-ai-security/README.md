## Prerequisites
- Module 1: AI Fundamentals
- Module 2: AI Threat Landscape
- Module 3: Offensive AI Security (recommended)

## Learning Objectives
After this module, you will be able to:
- Implement multi-layer input validation for LLM applications
- Design output filtering that catches sensitive data leakage
- Deploy guardrail frameworks (NeMo, Guardrails AI, custom)
- Apply least-privilege access control to AI systems
- Secure RAG pipelines against poisoning and exfiltration
- Sandbox AI agents to limit blast radius
- Protect model assets from theft and tampering
- Implement PII detection and data loss prevention for AI
