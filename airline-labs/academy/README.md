## 🚀 Getting Started

1. Start with **Module 1** if you're new to AI/ML concepts
2. Jump to **Module 3** if you already understand AI and want to see attacks
3. Go straight to **Module 5** if you want hands-on labs immediately
4. Start at **Module 6** if you're focused on governance and compliance

Each module is self-contained with theory, examples, and airline-specific scenarios.
